## üéØ Rumusan Masalah & Tujuan

### Masalah (Problem Statement)
Indonesia menghasilkan **500,000 ton minyak jelantah/tahun** dari sektor food service, namun tingkat daur ulang hanya **15-20%**. Akibatnya:
- ‚ùå Kontaminasi air & lahan
- ‚ùå Emisi karbon dari produksi minyak baru
- ‚ùå Pengumpul informal tanpa perlindungan sosial
- ‚ùå Kurangnya transparansi supply chain

### Rumusan Penelitian
**Bagaimana merancang platform digital yang terintegrasi untuk meningkatkan akses masyarakat terhadap pusat pengumpulan minyak jelantah dan meningkatkan kesadaran tentang daur ulang berkelanjutan?**

#### Sub-pertanyaan:
1. Bagaimana merancang user experience yang efektif untuk menemukan penampungan terdekat?
2. Bagaimana menyediakan konten edukatif yang komprehensif dan mudah dipahami?
3. Bagaimana sistem rating & review meningkatkan kepercayaan pengguna?

### Tujuan Penelitian

#### Tujuan Teknis
‚úÖ Membangun aplikasi mobile React Native dengan 5+ screens utama
‚úÖ Backend REST API dengan 15+ endpoints
‚úÖ Integrasi Google Maps untuk location-based services
‚úÖ Sistem rating & review untuk feedback pengguna

#### Tujuan Dampak
‚úÖ Meningkatkan aksesibilitas masyarakat ke penampungan minyak jelantah
‚úÖ Meningkatkan kesadaran tentang daur ulang berkelanjutan
‚úÖ Memberdayakan pengumpul melalui transparansi harga
‚úÖ Dukungan UN SDG 12 (Circular Economy) & SDG 4 (Quality Education)

### Manfaat Penelitian

**Akademis:**
- Kontribusi teoritis untuk sustainable development & digital inclusion
- Best practices full-stack development (Backend + Mobile + Maps API)
- Case study implementasi di negara berkembang
- 1,600+ lines production-ready code

**Praktis:**
- Platform siap pakai untuk entrepreneurs di circular economy
- Template untuk aplikasi waste management di daerah lain
- Peningkatan transparansi supply chain minyak jelantah
- Pemberdayaan ekonomi pengumpul informal

**Lingkungan:**
- Meningkatkan daur ulang minyak jelantah dari 15% ‚Üí 35%
- Mengurangi 150+ ton/tahun limbah ke landfill
- Menghemat 600+ ton/tahun crude oil production
- Pengurangan 1,500 ton CO2eq annual emissions

## üìö API Documentation

### Base URL
```
http://localhost:5000/api/v1
```

### Locations Endpoints

#### Get All Locations
```http
GET /locations?page=1&limit=20
```

Response:
```json
{
  "data": [
    {
      "id": 1,
      "name": "UD Maju Jaya",
      "address": "Jl. Sudirman No. 123",
      "city": "Jakarta Pusat",
      "latitude": -6.2088,
      "longitude": 106.8456,
      "phone": "021-1234567",
      "whatsapp": "6281234567890",
      "opening_hours": "09:00 - 17:00",
      "price_per_liter": 3000,
      "average_rating": 4.5,
      "reviews_count": 12
    }
  ],
  "total": 45,
  "page": 1,
  "limit": 20
}
```

#### Find Nearby Locations
```http
GET /locations/nearby?lat=-6.2088&lng=106.8456&radius=10
```

Finds all locations within 10 km radius.

#### Get Location Detail
```http
GET /locations/:id
```

### Tutorials Endpoints

#### Get All Tutorials
```http
GET /tutorials?page=1&limit=20
```

#### Get Tutorial Detail
```http
GET /tutorials/:id
```

#### Get Tutorials by Category
```http
GET /tutorials/category/environment
```

Categories: `environment`, `storage`, `cooking`, `recycling`, `health`

### Reviews Endpoints

#### Submit Review
```http
POST /reviews
Authorization: Bearer <JWT_TOKEN>
Content-Type: application/json

{
  "location_id": 1,
  "rating": 5,
  "review_text": "Penampungan yang bagus!"
}
```

#### Get Reviews for Location
```http
GET /reviews/location/:id?page=1&limit=10
```
